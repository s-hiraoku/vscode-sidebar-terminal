# インストール

## VS Code Marketplace

最も簡単なインストール方法:

1. VS Codeを開く
2. 拡張機能ビューへ移動（`Ctrl+Shift+X` / `Cmd+Shift+X`）
3. **"Secondary Terminal"** を検索
4. **インストール** をクリック

またはコマンドラインから:

```sh
code --install-extension s-hiraoku.vscode-sidebar-terminal
```

## Open VSX Registry

VSCodium、Gitpod、Eclipse Theiaユーザー向け:

- [Open VSX](https://open-vsx.org/extension/s-hiraoku/vscode-sidebar-terminal) からインストール

## 必要条件

- **VS Code** 1.106.0 以降
- **Node.js** 18.0 以降（開発時のみ）
- **Windows**、**macOS**、**Linux** で動作

## インストール確認

インストール後、VS Codeのアクティビティバー（左サイドバー）にターミナルアイコン（ST）が表示されます。クリックして最初のターミナルを開きましょう。
