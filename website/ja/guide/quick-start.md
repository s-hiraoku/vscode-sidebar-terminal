# クイックスタート

1分以内でSecondary Terminalを使い始めましょう。

## 最初のターミナルを開く

1. アクティビティバー（左サイドバー）の **ターミナルアイコン（ST）** をクリック
2. デフォルトシェルでターミナルが起動
3. 通常通りコマンドを入力

## 基本操作

| アクション | ショートカット |
|--------|----------|
| 新規ターミナル | ヘッダーの **+** ボタンをクリック |
| ターミナルを閉じる | `Ctrl+Shift+X` / `Cmd+Shift+X` |
| ターミナル切り替え | タブをクリック、または `Alt+1..5` / `Cmd+Alt+1..5` |
| ターミナル分割 | `Ctrl+Shift+5` / `Cmd+\` |
| ターミナル内検索 | `Ctrl+F` / `Cmd+F` |
| ターミナルクリア | `Ctrl+K` / `Cmd+K` |
| コピー | `Ctrl+C` / `Cmd+C`（テキスト選択時） |
| 貼り付け | `Ctrl+V` / `Cmd+V` |

## AIエージェント連携を試す

対応するAI CLIエージェントを実行:

```sh
# Claude Code
claude

# GitHub Copilot
copilot

# Gemini CLI
gemini

# Codex CLI
codex
```

エージェントが検出されると、ターミナルヘッダーに **ステータスインジケーター** が表示されます。

### AIエージェント向けショートカット

| アクション | ショートカット |
|--------|----------|
| 現在のファイルパスを挿入 | `Ctrl+Alt+L` / `Cmd+Alt+L` |
| 全オープンファイルを挿入 | `Ctrl+Alt+A` / `Cmd+Alt+A` |
| スクリーンショット貼り付け（macOS） | `Cmd+V` |
